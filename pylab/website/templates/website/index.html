Python Workshop

The main purposes of Python Workshops are:

  • learn to code in Python
  • share programming experience with other participants
  • create useful and valuable projects
  • show the devotion to Python programming language and open source
Everyone can attend regular meetings. We have no restrictions on age, sex, nationality or knowledge level. It is possible to meet a beginner as well as a mature developer on Python workshops. Participation in any event is absolutely free. You are welcome!
